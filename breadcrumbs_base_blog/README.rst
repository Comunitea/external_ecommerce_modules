Blog extension for Breadcrumbs Base module
------------------------------------------

This module adds friendly multi-level breadcrumbs bar in the blog and post pages.

    **Attention!**

    This module removes default breadcrumb bar from the blog and post pages.

This module uses default Bootstrap elements.
